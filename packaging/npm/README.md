# kervo

**Stop re-explaining your project to AI. `kervo init` once.**

kervo compiles your repository into a deterministic Context Artifact and
injects it into `CLAUDE.md` — facts extracted deterministically,
interpretations only as trust-labeled proposals.

This npm package is the npx wrapper for the kervo binary. At v0.0.x it
prints install instructions; binary bootstrap ships with the first tagged
release.

Source, docs, and issues: **https://github.com/kervo-os/kervo**
